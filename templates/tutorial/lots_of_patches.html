Lots of Patches

In this third part of the interactive cookbook, you'll find a number of well-documented Sporth patches that can be run in the browser. Note that there are a few other places where you can find patches:

  • The AudioMasher collection has a growing number of patches, also generally meant to be played in-browser. You can easily add your own patches there.
  • The original cookbook has some additional patches that aren't in the interactive version, because they rely on external files. Try them out using the native version of Sporth.
  • The examples included with the Sporth distribution demonstate a variety of Sporth ugens.
